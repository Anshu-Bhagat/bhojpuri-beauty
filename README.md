# 💖 Romantic Bhojpuri Song Web Gift

A creative and emotional **Bhojpuri song-based web project** built using **HTML, CSS, and JavaScript**. This project delivers a digital gift experience with beautiful animations and synced song lyrics.

---

## ✨ Features

- 💕 Animated welcome screen with floating hearts  
- 🎁 Interactive “Gift” button  
- 🎵 Bhojpuri song with auto-play  
- 📝 Synchronized lyrics display  
- 🎨 Modern UI with glassmorphism effect  
- 📱 Responsive design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Animations & UI Design)  
- JavaScript (DOM Manipulation & Audio Sync)  

---

## 📂 Project Structure
    ┣ 📄 index.html # Gift landing page
    ┣ 📄 song.html # Song + lyrics page
    ┣ 📁 assets
    ┃ ┗ 🎵 song.mp3 # Audio file


---

## 🚀 How to Run

1. Download or clone the repository  
2. Make sure `song.mp3` is inside the `assets` folder  
3. Open `index.html` in your browser  
4. Click on **YES** and enjoy the experience 💖  

---
